# GOLS logo assets

Approved PNG files for **Game On Live Studio**. Do not rename, edit, stretch, rotate, or recolor.

| File | Use on |
|------|--------|
| `gols-logo-primary-red.png` | Icons, favicons, compact headers, watermarks |
| `gols-logo-secondary-black-text.png` | White / light grey backgrounds |
| `gols-logo-secondary-white-text.png` | Black, Zodiac Blue, dark photos |
| `gols-logo-secondary-all-white.png` | Dark backgrounds where red mark competes |
| `gols-logo-secondary-all-black-reference.png` | Internal layout reference only |
