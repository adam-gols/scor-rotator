# Start prompt (optional)

Onboarding starts **automatically** when you open Agent chat in Cursor — you usually do not need this file.

Use only if hooks are disabled or onboarding did not start:

```text
Start my new project — interview me, create the repo, and bootstrap when I approve.
```

Legacy explicit prompt:

```text
Read AGENTS.md and docs/onboarding-interview.md.
Run the onboarding interview: ask me questions in small batches, write answers into docs/bootstrap-spec.md and docs/project-context.md, create the git repo, show a summary, and wait for my approval before bootstrapping.
```
